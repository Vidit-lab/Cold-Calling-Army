🔧 Setup & Quick Start

One-Time Setup

  1. Run supabase_schema.sql in Supabase → SQL Editor
  2. In Coolify → your resource → Environment Variables, set all keys (LiveKit, Gemini, Vobiz, Supabase) — never in the UI
  3. Create the SIP trunk once: POST /api/setup/trunk (or set OUTBOUND_TRUNK_ID directly), then redeploy
  4. ✏️ AI Prompt → customise for your business → Save
  5. 🤖 Agents → create at least one profile
  6. 📞 Single Call → test with your own number

🔒 API keys are configured only via server environment variables — the dashboard never displays or accepts them.
